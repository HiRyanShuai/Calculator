# Calculator

A calculator project from the Odin project. It supports positive and negative signs, decimal points, backspace, and can display the current formula.

The difficulty to determine the input states (such as determining which operand is currently being input, continuously inputting equal signs, etc.), and adjust the data logic according to the state.

Problems to be solved: When the number is too long, it will overflow the display area; the calculation accuracy problem of javascript.

To be improved: keyboard support.
